# Dining-Philosophers
Dining Philosophers Problem simulated in C++ 

# Prerequisites & Versions
`Ubuntu` **18.04**

`g++` **7.4.0**

`C++` **14**

`ncurses` **5**


# Build
`make`

# Run
`./dining_philosophers`

# Keybinds

- **q** - quit program
- **a** - add one philosopher
- **z** - take one philosopher away
